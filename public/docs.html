Play Store Reviews Manager Documentation

Endpoints

  • POST /api/exchange — Exchange auth code for tokens
  • POST /api/refresh — Refresh access token
  • GET /api/reviews?pkg=[PACKAGE] — Fetch reviews
  • GET /api/live — Health check; returns {"status":"live"}

cURL Examples

curl -X POST https://<domain>/api/exchange \
  -H "Content-Type: application/json" \
  -d '{"code":"CODE","client_id":"ID","client_secret":"SECRET"}'

curl -X POST https://<domain>/api/refresh \
  -H "Content-Type: application/json" \
  -d '{"refresh_token":"RT","client_id":"ID","client_secret":"SECRET"}'

curl "https://<domain>/api/reviews?pkg=com.example.app" \
  -H "Authorization: Bearer ACCESS_TOKEN"

curl https://<domain>/api/live

Setup Guide

  1. Enable Android Publisher API in Google Cloud.
  2. Create OAuth Client ID with redirect URI postmessage.
  3. Use the generated Client ID & Secret in the UI.
