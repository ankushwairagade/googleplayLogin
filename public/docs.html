Play Store Reviews Manager Documentation

Endpoints

  • POST /api/exchange — exchange auth code for tokens.
  • POST /api/refresh — refresh access token.
  • GET /api/reviews?pkg=[PACKAGE] — fetch reviews.
  • GET /api/live — health check, returns {"status":"live"}.

Google Cloud Setup

  1. Visit the Google Cloud Console.
  2. Create/select a project and enable the Android Publisher API.
  3. Under APIs & Services > Credentials, create an OAuth client ID:
    • Application type: Web application
    • Authorized redirect URIs: postmessage
  4. Copy your Client ID & Client Secret.

Using the App

  1. Open index.html, enter Client ID & Secret, then Initialize OAuth.
  2. Click Sign In to grab tokens, or Refresh Token later.
  3. Enter your app package name, then Fetch Reviews.
